# 사용법

1. csv 파일을 업로드 하여 회원 목록을 추가 한다
2. 입력 모드에서 상품을 추가한다
EX)
이모 빌딩, 10
이모 벤츠, 30
이모 냉장고, 10
이모 세탁기, 5
3. 입력이 끝났으면 추첨모드로 들어가서 추첨을 한다.

# 주의

- 사파리는 안될수도 있다
- 버그가 있을수도 있다
